Challenge-Clothes modify app

  • Category: UX/UI Web design
  • Client: Code Academy
  • Project date: 01 March, 2021
  • Project URL: Private

In nowaday people more like buying clothes online,but not all clothes have likable parts in them or don't fit well.The clothes modifying system can solve this problem , by letting people modify clothes by them wishes.

01 Discover

To find out the needs of women, we used an online survey and interview were conducted which found that the biggest problem faced by women is the size of clothes.

02 Define

To define the person for whom this app is being developed, i created persons to imagine a typical user of this app. All the most common needs and desires of this person are based on the results of surveys.

03 User yourney map

User journey maps drawn to detect user-pain points and represent the user’s path to the desired result.

04 Ideate

When we trying to find the right solution, we use sketches. This helps to find inspiration and outline the elements that will potentially be used in the final product.
